Introduction

Object-Oriented Programming (OOP) allows you to organize your VB.NET Windows Forms projects into clear, reusable components. This module covers the 3 pillars of OOP — Inheritance, Encapsulation, and Polymorphism — using real student-related examples.

0. Kinds of Inheritance in VB.NET

VB.NET mainly supports **Single**, **Multilevel**, and **Hierarchical** Inheritance.

✔ 1. Single Inheritance

One child inherits from one parent.

Public Class Student
    Public Property FullName As String
End Class

Public Class SeniorHighStudent
    Inherits Student
    Public Property Strand As String
End Class

✔ 2. Multilevel Inheritance

A child inherits from a class that also inherits from another class (Grandparent → Parent → Child).

Public Class Person
    Public Property Name As String
End Class

Public Class Student
    Inherits Person
    Public Property StudentID As String
End Class

Public Class AthleteStudent
    Inherits Student
    Public Property Sport As String
End Class

✔ 3. Hierarchical Inheritance

Many child classes inherit from the same parent class.

Public Class Student
    Public Property Name As String
End Class

Public Class HonorStudent
    Inherits Student
    Public Property HonorRank As String
End Class

Public Class WorkingStudent
    Inherits Student
    Public Property WorkHours As Integer
End Class

1. Inheritance — Student Examples

Inheritance lets a child class reuse fields and methods from a parent class.

Public Class Student
    Public Property StudentID As String
    Public Property FullName As String
    Public Property Age As Integer

    Public Sub ShowBasicInfo()
        MessageBox.Show("Name: " & FullName & vbCrLf & "ID: " & StudentID)
    End Sub
End Class

Public Class SeniorHighStudent
    Inherits Student
    Public Property Strand As String
End Class

Public Class CollegeStudent
    Inherits Student
    Public Property Course As String
End Class

' Usage:
Dim sh As New SeniorHighStudent()
sh.FullName = "Juan Dela Cruz"
sh.StudentID = "2025-SHS-001"
sh.Strand = "STEM"
sh.ShowBasicInfo()

2. Encapsulation — Protecting Student Data

Encapsulation protects sensitive values by using private fields with validation.

Public Class StudentGrade
    Private _grade As Double

    Public Property Grade As Double
        Get
            Return _grade
        End Get
        Set(value As Double)
            If value >= 0 And value <= 100 Then
                _grade = value
            Else
                MessageBox.Show("Grade must be between 0 and 100.")
            End If
        End Set
    End Property
End Class

3. Polymorphism — Student-Focused

Polymorphism allows the same method to behave differently depending on the object type.

Public Class Student
    Public Overridable Function ShowStudentType() As String
        Return "General Student"
    End Function
End Class

Public Class SeniorHighStudent
    Inherits Student
    Public Overrides Function ShowStudentType() As String
        Return "Senior High Student"
    End Function
End Class

Public Class CollegeStudent
    Inherits Student
    Public Overrides Function ShowStudentType() As String
        Return "College Student"
    End Function
End Class

Full Combined Example

Public Class Student
    Public Property FullName As String
    Public Property StudentID As String

    Public Overridable Sub ShowInfo()
        MessageBox.Show("Student: " & FullName)
    End Sub
End Class

Public Class SeniorHighStudent
    Inherits Student

    Private _strand As String
    Public Property Strand As String
        Get
            Return _strand
        End Get
        Set(value As String)
            _strand = value
        End Set
    End Property

    Public Overrides Sub ShowInfo()
        MessageBox.Show("SHS: " & FullName & " - Strand: " & Strand)
    End Sub
End Class

Public Class CollegeStudent
    Inherits Student

    Private _gpa As Double
    Public Property GPA As Double
        Get
            Return _gpa
        End Get
        Set(value As Double)
            If value >= 1.0 And value <= 5.0 Then
                _gpa = value
            Else
                MessageBox.Show("GPA must be between 1.0 and 5.0")
            End If
        End Set
    End Property

    Public Overrides Sub ShowInfo()
        MessageBox.Show("College: " & FullName & " - GPA: " & GPA)
    End Sub
End Class

' Usage in Form:
Dim s As Student = New CollegeStudent()
s.FullName = "Ana Santos"
CType(s, CollegeStudent).GPA = 1.75
s.ShowInfo()
